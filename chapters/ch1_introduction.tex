\section{Introduction}
\label{sec:ch1}

Classical Ramsey Theory asks for the minimum number of vertices required to guarantee the existence of a clique or an independent set in a graph. We refer to these maxima as \textit{Ramsey numbers}.

\begin{definition}
  \label{def:ramsey-number}
  The \textit{Ramsey number} $R(s, t)$ is the minimum number $N$ such that any graph on $N$ vertices contains a clique of size $s$ or an independent set of size $t$.
\end{definition}

The existence of these numbers was proven by Ramsey \cite{ramsey1930} in 1930, but their quantitative behaviors remain largely unknown despite decades of intensive research. In particular, it is known that $R(3, 3) = 6$ and $R(4, 4) = 18$, but the exact values of $R(5, 5)$ and beyond are yet to be determined. The cornerstone of Ramsey Theory due to Erdős and Szekeres in 1935 gives the following upper bound on $R(s, t)$:

\begin{theorem}[Erdős--Szekeres \cite{erdos1935some}]
  \label{thm:erdos-szekeres}
  For $s, t \geq 2$,
  \[
    R(s, t) \leq \binom{s + t - 2}{s - 1}.
  \]
\end{theorem}

Following the fact that $\binom{2t - 2}{t - 1} < 2^{2t}$, we can see $R(t, t) < 4^{t}$. Very little progress had been made on this bound until the recent breakthrough of Campos, Griffiths, Morris, and Sahasrabudhe \cite{campos2025exponentialimprovementdiagonalramsey}, who gave the first exponential improvement since the result of Erdős and Szekeres. On the other hand, the lower bound of ramsey numbers has been even more challenging to determine. A rather basic result which already gives the best known exponents is the famous random graph construction due to Erdős in 1947, marking one of the first applications of the probabilistic method in combinatorics. Before stating the result, we first define the \textit{Erdős--Rényi random graph}.

\begin{definition}
  \label{def:random-graph}

  For $n \geq 1$ and $p \in [0, 1]$, the \textit{random graph} $G_{n, p}$ is the graph on $n$ vertices in which each of the $\binom{n}{2}$ edges is present independently with probability $p$.
\end{definition}

\textit{Remark: For constant $p$, the random graph $G_{n, p}$ is dense in the sense that it has $\Theta(n^2)$ many edges almost surely.}

\begin{theorem}[Erdős \cite{erdos1972ramsey}]
  \label{thm:diagonal-ramsey}
  For $t \geq 3$,
  \[
    \sqrt{2}^t < R(t, t) < 4^t.
  \]
\end{theorem}

\begin{proof}
  The upper bound follows from Theorem \ref{thm:erdos-szekeres}. For lower bound, set $N = \lfloor \sqrt{2}^t \rfloor$ and consider the random graph $G_{N, 1/2}$. The expected number of cliques of size $t$ is
  \[
    \binom{N}{t} \cdot 2^{-\binom{t}{2}} < \frac{N^t}{t!}2^{-\binom{t}{2}} \leq \frac{\sqrt{2}^{t^2}}{t!}2^{-\binom{t}{2}} = \frac{\sqrt{2}^t}{t!} = \frac{\sqrt{2}^{t - 3}}{t(t - 1)\cdots 4} \cdot \frac{\sqrt{2}^3}{3!} \leq \frac{\sqrt{2}^3}{3!} < \frac{1}{2}.
  \]
  Similarly, the expected number of independent sets of size $t$ is less than $1/2$. It now follows that there exists a graph on $N$ vertices containing neither a clique nor an independent set of size $t$.
\end{proof}

The above strategy may be optimized to give a lower bound of $\frac{{t}}{e\sqrt{2}}\sqrt{2}^t$, which has been astonishingly resilient to improvement. The only noteworthy improvement since 1947 is achieved in 1977 by Spencer \cite{Spencer1975RamseysT}, who used the Lovász local lemma to raise the bound to $(1 - o(1))\frac{t\sqrt{2}}{e}\sqrt{2}^t$, an improvement by a factor of 2. For further background on Ramsey theory, see Conlon, Fox, and Sudakov \cite{conlon2015recentdevelopmentsgraphramsey}.

From Theorem \ref{thm:diagonal-ramsey}, we can see that any graph on $n$ vertices contains a clique or independent set of size at least $\frac{1}{2}\log_2 n$, whereas there exists a graph on $n$ vertices containing neither a clique nor an independent set of size $2\log_2 n$. Apart from determining the bounds of Ramsey numbers, the search for explicit graph constructions with logarithmically small cliques and independent sets has also been a major open problem in Ramsey Theory. Despite decades of intensive research, all known constructions still rely on some randomness. This thus motivates the question of whether graphs with small cliques and independent sets inherently resemble dense random graphs like $G_{n, 1/2}$; we refer to such graphs as \textit{Ramsey graphs}.

\begin{definition}
  \label{def:ramsey-graph}

  Let $C > 0$ be a constant. A \textit{$C$-Ramsey graph} is a graph with no clique or independent set of size greater than $C\log n$.
\end{definition}

It is useful to also define a shorthand for cliques and independent sets.

\begin{definition}
  \label{def:hom}

  A set $S \subseteq V(G)$ is \textit{homogeneous} if it induces a clique or an independent set in a graph $G$.
\end{definition}

The belief that Ramsey graphs and random graphs are intimately related is supported a number of theorems showing that certain ``richness'' properties of random graphs also hold for Ramsey graphs. The first result that cemented this connection is due to Erdős and Szemerédi, showing that Ramsey graphs are also dense.

\begin{theorem}[Erdős--Szemerédi \cite{erdos1972ramsey}]
  \label{thm:ramsey-density}

  Let $d \in [0, 1]$ and let $G$ be a graph on $n$ vertices with edge density $d$. Then $G$ contains a homogeneous set of size greater than $K = \frac{c}{d\log(1/d)} \log n$ for some constant $c > 0$ and sufficiently large $n$.
\end{theorem}

  Suppose not. By the pigeonhole principle, there exists $m \geq n/2$ vertices in $G$ with degree at most $2dn$. We work with the graph $G'$ induced by these $m$ vertices. Let $I$ be the largest independent set in $G'$ and let $k = |I|$. Note that $\sum_{v \in I} \deg(v) \leq 2dnk \leq 4dmk$. Let $s = 8dk$. Again by the pigeonhole principle, there are at least $m/2 - k$ vertices $v \in V(G') \setminus I$ with $\deg_{I}(v) \leq s$. Let $J$ be the set of these vertices. Since $k < K$,

    s < 8dK = \frac{c \log n}{\log(1/d)}

\begin{corollary}
  \label{cor:ramsey-density}
  For any $C$-Ramsey graph $G$, there exists $\delta = \delta(C) \in (0, 1/2)$ such that the edge density of $G$ lies in the interval $[\delta, 1-\delta]$.
\end{corollary}

\begin{proof}
  Apply Theorem \ref{thm:ramsey-density} to $G$ and $\overline{G}$. 
\end{proof}

Due to the rich variety of induced subgraphs in dense random graphs, a popular angle to study the pseudorandom properties of Ramsey graphs is to consider their induced subgraphs. Some notable results are a theorem of Shelah \cite{shelah1997erdhosrenyiconjecture} which asserts that every $n$-vertex Ramsey graph contains $2^{\Omega(n)}$ non-isomorphic induced subgraphs, and a theorem of Prömel and Rödl \cite{PROMEL1999379} which shows the existence of $\delta = \delta(C) > 0$ such that every $C$-Ramsey graph on $n$ vertices contains an induced copy of all graphs on at most $\delta \log n$ vertices. 

It is also interesting to look at the edge statistics of Ramsey graphs. For a graph $G$, let 
\[
  \Phi(G) \coloneqq \{e(H) : H \text{ is an induced subgraph of } G\}
\]
Erdős and McKay \cite{EFS} conjectured that for any Ramsey graph $G$ on $n$ vertices, there exists $\delta = \delta(C) > 0$ such that
\[
  \{1, 2, \ldots, \delta n^2\} \subseteq \Phi(G).
\]
It was proven for Random graphs by Calkin, Frieze, and Mckay \cite{erdosmckayrandomgraph}, but the conjecture for Ramsey graphs remained open for decades and only started to show significant progress in recent years. In particular, a chain of results were sparked after the work of Narayanan, Sahasrabudhe, and Tomon \cite{narayanan2017ramseygraphsinducesubgraphs} establishing that $|\Phi(G)| = n^{2 - o(1)}$ for Ramsey graph $G$ on $n$ vertices. Their proof builds on some ideas from the work of Bukh and Sudakov \cite{BUKH2007612} which showed that every Ramsey graph contains a linear sized subgraph with many distinct degrees, a feature that dense random graphs also exhibit. Kwan and Sudakov \cite{kwan2021ramseygraphsinducesubgraphs} later improved the bound to $|\Phi(G)| = \Omega(n^2)$ by refining the argument of Narayanan, Sahasrabudhe, and Tomon. Eventually, Kwan, Sah, Sauermann, and Sawhney \cite{Kwan_2023} closed the conjecture with a fairly involved approach using tools from Fourier analysis, random matrix theory, and the theory of Boolean functions. 

This dissertation will start by studying the result of Bukh and Sudakov \cite{BUKH2007612} on large induced subgraphs in Ramsey graphs with many distinct degrees. This work not only tightens the connection between Ramsey graphs and random graphs, but also introduces a key feature of Ramsey graphs that unlocks the subsequent $|\Phi(G)| = \Omega(n^2)$ result by Kwan and Sudakov, which we will present in the following chapter. Lastly, we go through the broad strokes of the proof to the Erdős--McKay conjecture by Kwan, Sah, Sauermann, and Sawhney \cite{Kwan_2023}, touching on the core ideas of their approach.

\subsection{Notations and Tools}

We call $e(G)$ the \text{size} of a graph $G$. We omit all floor and ceiling signs when they are not crucial for simplicity.

\begin{proposition}[Turán's theorem]
  \label{prop:turan}
  
  Every graph $G$ with $n$ vertices and $m$ edges contains an independent set of size at least $n^2/(2m + n)$.
\end{proposition}

\begin{proposition}[Hoeffding's inequality]
  \label{prop:azuma-hoeffding}
  
  Let $X_1, X_2, \ldots, X_n$ be independent real-valued random variables with $0 \leq X_i \leq 1$ for each $i$, and write $X = \sum_{i = 1}^n X_i$. Then for $t \geq 0$, 
  \[
    \prob(|X - \E[X]| \geq t) \leq 2e^{-2t^2/n}.
  \]
\end{proposition}

\begin{proposition}
  \label{prop:hypergeometric}
  
  Let $X$ denote the number of successes in $D$ draws, without replacement, from a population of size $N$ which contains $M$ successes. Then $\E[X] = DM/N$ and for $t \geq 0$, 
  \[
    \prob(\left|X - \E[X]\right| \geq t) \leq 2e^{-2t^2D}.
  \]
\end{proposition}
